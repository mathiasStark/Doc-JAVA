ACTIVE RECORD -> É UM PADRÃO DE PROJETO ONDE A CLASSE ENTIDADE É DIRETAMENTE LIGADA A  TABELA NO BANCO DE DADOS.
A CLASSE JÁ POSSUI OS PRÓPRIOS MÉTODOS DE ACESSO AO BANCO COMO SAVE, FIND DELETE ETC. GERALMENTE ESTES MÉTODOS SÃO HERDADOS DE UMA OUTRA CLASSE CHAMADA ACTIVE RECORD.

CLIENTE C = NEW CLIENTE();
C.SAVE();
C.DELETE();
CLIENTE.FIND(); //METODO STATIC

DATA MAPPER -> PADRÃO UTILIZADO PELO JPA. SÃO UTILZIADOS METADADOS PARA MAPEAR OS ATRIBUTOS DA TABELA PARA O OBJETO NO JAVA. EM RESUMO É O MAPEAMENTO DO OBJETO QUE SERÁ UTILIZADO NO BANCO DE DADOS.
SERVE PARA IDENTIFICAR ONDE VAI SER PERSISTIDO CADA INFORMAÇÃO NAS COLUNAS DA TABELA.
