BAIXAR DRIVER DE CONEXAO PARA O DEVIDO BANCO:
EX.: driver jdbc mysql ZIP (mysql-connector-j-8.0.31.zip) 

CRIAR UMA PASTA LIBS DENTRO DO PROJETO JAVA, DESCOMPACTAR O ZIP E COLOCAR O ARQUIVO (mysql-connector-j-8.0.31.jar) DENTRO DESTA PASTA LIB.


ADICIONAR O ARQUIVO JAR AO BUILD
CLICAR COMM O BOTÃO DIREITO NO ARQUIVO JAR, BUILD PATH -> ADD TO BUILD PATH.

SETAR O CERTIFICADO DO SERVER:
JDBC:MYSQL://LOCALHOST?VERIFYSERVERCERTIFICATE=FALSE

USAR CONEXAO SEGURA SSL
JDBC:MYSQL://LOCALHOST?VERIFYSERVERCERTIFICATE=FALSE&USESSL=TRUE
